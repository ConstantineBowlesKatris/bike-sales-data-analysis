# 🚴 Bike Sales Data Analysis Dashboard (Excel)
Interactive Excel dashboard analyzing customer demographics and bike purchasing behavior using Pivot Tables, Pivot Charts, and Slicers.
<img width="1359" height="968" alt="image" src="https://github.com/user-attachments/assets/d2e7592d-b007-4f77-a4c4-363e43770f3c" />
## Business Problem

A bicycle retailer wants to better understand which customer characteristics are associated with bike purchases in order to improve marketing campaigns and customer targeting.

Using Excel, I analyzed demographic and socioeconomic data to identify purchasing patterns and present the results in an interactive dashboard.

---

## Dashboard Features

- Interactive slicers for filtering data
- Customer segmentation analysis
- Income comparison
- Age group analysis
- Commute distance analysis
- Regional comparisons
- Gender and marital status breakdowns
- Dynamic Pivot Charts

---

## Excel Skills Demonstrated

- Pivot Tables
- Pivot Charts
- Interactive Slicers
- Data Cleaning
- Data Transformation
- Dashboard Design
- Excel Chart Formatting

---

## Dataset

The dataset contains customer information including:

- Marital Status
- Gender
- Income
- Number of Children
- Education
- Occupation
- Home Ownership
- Number of Cars
- Commute Distance
- Region
- Age
- Bike Purchase (Yes/No)

---

## Business Insights

This dashboard enables users to:

- Identify customer segments most likely to purchase a bike
- Compare purchasing behavior across regions
- Analyze the relationship between income and purchasing decisions
- Explore how age, education, occupation, and commute distance affect buying behavior
- Support data-driven marketing decisions

## How to Use

1. Download the Excel workbook.
2. Open it in Microsoft Excel.
3. Navigate to the **Dashboard** worksheet.
4. Use the slicers to interactively filter the visualizations and explore customer purchasing trends.
